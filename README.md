# Animated Dashboard Card

A challenge designed to test practical understanding of Transitions, Transforms, and Animations in Tailwind CSS.

## Features
- Semantic HTML5 structure
- Smooth hover and loading animations
- Interactive scaling and shadow effects on cards
- Nested element animations using `group` and `group-hover`
- Performance-conscious transitions
- Optional “Loading Complete” animation with bounce or fade effects

## Tech Stack
- HTML5
- Tailwind CSS
- JavaScript (for optional loading completion animation)

## Purpose
Practical application of Transitions, Transforms, and Animations in Tailwind CSS to create interactive, responsive UI components with smooth motion effects — without relying on external libraries.

## How to Use
Download the project files and run `npm run dev` in the terminal  
or  
Open the live link associated with this folder.
